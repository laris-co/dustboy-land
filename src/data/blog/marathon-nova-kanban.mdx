## ไฮไลต์

- **cross-engine kanban** — codex (gpt-5.5) + glm-5 ทำงานบน board เดียวแบบไม่ผ่าน GitHub ออก 3 deliverable เข้า main
- **honest eval R² = 0.70** — เลขที่ซื่อสัตย์ (LODCV) ดีกว่า 0.86 ที่รั่ว
- **Nova chain rescue** — ไล่ `safe_l2=0` จนขยับ
- ถือเส้น **security boundary** เรื่อง private key / การโอนเงิน ทุกรูปแบบการขอ
